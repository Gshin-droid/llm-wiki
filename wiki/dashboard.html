Claude Code changelog v2.1.216–2.1.217 — sandbox.filesystem.disabled
Автономный разведчик: новая настройка отключает только файловый слой Bash-сэндбокса, оставляя сетевую изоляцию в силе — точнее прежнего бинарного выбора "сэндбокс целиком / выключить целиком". Официальная документация прямо называет риск (команда может расширить себе доступ на будущее) и разрешает включать только из user/managed settings. Плюс настраиваемые лимиты параллелизма субагентов и фикс `--max-budget-usd` для фоновых субагентов.
