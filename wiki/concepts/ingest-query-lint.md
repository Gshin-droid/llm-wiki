# Ingest / Query / Lint (три операции вики)

## Определение
Три базовые операции, которыми LLM управляет персональной вики в рамках [[persistent-wiki-pattern]].

## Ingest (загрузка)
Пользователь даёт новый источник → LLM читает его, обсуждает ключевые выводы, пишет саммари в `wiki/sources/`, обновляет/создаёт страницы сущностей и концептов, обновляет индекс. Один источник может затронуть 10–15 страниц.

## Query (запрос)
Пользователь задаёт вопрос → LLM читает индекс, находит релевантные страницы, синтезирует ответ с цитатами на `[[файлы]]`. Ценные ответы сохраняются как отдельные страницы в `wiki/synthesis/`, чтобы не исчезать в истории чата.

## Lint (очистка)
Периодическая проверка здоровья вики: противоречия между страницами, устаревшие утверждения, страницы-сироты без входящих ссылок, важные концепты без своей страницы, битые перекрёстные ссылки, пробелы в данных.

[[karpathy-jarvis-personal-ai-memory]]

Реализовано как раздел "Операции" в [[CLAUDE.md]] этого vault'а, с добавлением четвёртой операции — Project (см. [[jarvis-personal-wiki]]), т.к. пользователь ведёт рабочие ИИ-проекты внутри той же вики.
