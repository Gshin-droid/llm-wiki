# Чек-лист "сайт за $10 000" (через Claude Code)

## Определение
Воркфлоу и 8-пунктовый чек-лист из [[metics-media-10k-website]] для получения кастомного, немассового результата от Claude Code при вёрстке сайта — вместо типового "AI-шаблонного" вида.

## 8 пунктов чек-листа
Point of view, typography, colour, hierarchy, imagery, motion, mobile (дизайн под телефон, не просто адаптив), invisible stuff (то, что делает сайт быстрым и завершённым). Делятся на три группы: вкус, содержание, ощущаемое качество.

## Ключевые шаги воркфлоу
1. Claude Code в десктоп-приложении, без настройки под разработчика
2. Установить два [[claude-skills|скила]]: **Frontend Design** (Anthropic, банит заезженные шрифты типа Inter) + **UI/UX Pro Max** (57 стилей, 95 палитр)
3. Референсы — скриншоты понравившихся сайтов (Dribbble, Awwwards, Pinterest), а не попытка описать вкус словами
4. Сильный бриф + явная просьба "задай уточняющие вопросы" — момент, который определяет весь результат
5. Изображения/видео — ChatGPT для фото, агрегаторы (например, ElevenLabs) для видео из фото; Claude сам пишет промт для генерации
6. Готовые компоненты с 21st.dev — Claude может отказаться встраивать React-компонент, если проект статический (архитектурная защита, не слепое выполнение)
7. **Polish pass**: сверка с чек-листом → батч правок по ощущению ("более рукотворно", не конкретный список фич) → точечные микро-анимации на "плоских" секциях → замена заезженных AI-шрифтов (Inter → что-то другое) → отдельный проход по мобильной версии
8. Деплой: хостинг + собственный домен; для статических сайтов достаточно простого тарифа, для сайтов с бэкендом/логином — тариф уровнем выше
