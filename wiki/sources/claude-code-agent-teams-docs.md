## Саммари
Полная официальная документация по Agent Teams — оркестрации нескольких полных инстансов Claude Code (lead + teammates с прямой связью друг с другом, общим task list и mailbox). Прочитана целиком, чтобы закрыть пробел вики: до этого Agent Teams упоминались парой предложений в [[claude-code]]. Выжимка правил и архитектуры — в новом концепте [[agent-teams]].

## Ключевые факты
- Экспериментальная фича, требует `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1`; команды создаются неявно при спавне первого тиммейта (с v2.1.178, `TeamCreate`/`TeamDelete` удалены — это уточняет запись в [[claude-code-changelog-snapshot-2026-07]]).
- Главное отличие от субагентов: тиммейты общаются напрямую и самокоординируются через общий task list; субагенты только отчитываются вызывающему. Токены: teams значительно дороже (каждый тиммейт — полный инстанс).
- Официальная зона применимости: research/review, независимые модули, дебаг конкурирующими гипотезами, cross-layer. Антипаттерны: последовательные задачи, один файл, много зависимостей.
- Security-модель: сообщения между агентами помечены как «не от пользователя», тиммейт не может одобрять permissions и не может обходить отказы через другого тиммейта; permission-режим наследуется от lead'а в момент спавна (включая bypass!).
