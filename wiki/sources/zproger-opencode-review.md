**Дата загрузки:** 2026-07-10

## О чём источник
Практический обзор [[opencode]] — open-source CLI-агента для кодинга, позиционируемого автором как бесплатная альтернатива [[claude-code]].

## Ключевые тезисы
- **Мультипровайдерность** — ~100 провайдеров моделей "из коробки": облачные (OpenRouter, Perplexity, Anthropic по API), локальные (Ollama, LM Studio, Hugging Face). В отличие от Claude Code, который работает только с моделями Claude.
- **Три способа запуска**: терминал, десктоп-приложение (GUI), веб-версия на localhost — можно поднять на одной машине и управлять с телефона/другого ПК в локальной сети.
- **Встроенные режимы Plan / Build** — аналог Plan Mode в Claude Code: сначала план, затем реализация по нему.
- **`opencode.json`** — конфиг проекта; ключевая рекомендация автора — включить `ask` на любое редактирование файлов и bash-команды по умолчанию (защита от галлюцинаций типа "агент удалил весь проект"), похоже на дефолтный permission-режим Claude Code.
- **`AGENTS.md`** — файл-инструкция для самого агента (аналог `CLAUDE.md`): фиксирует стиль интерфейса, правило "не удалять существующие комментарии" и т.п., чтобы результат был последовательным между сессиями.
- **Skills и MCP** — OpenCode поддерживает ту же структуру папок, что Claude Code: skill из GitHub можно установить командой, ориентированной на Claude Code ("выбираем Claude Code и ставим skill"), автор использовал это напрямую. MCP (пример — Playwright для управления браузером естественным языком) подключается через `config`.
- Экономия токенов — если заранее указать агенту конкретный файл вместо описания всей задачи текстом, он не анализирует весь контекст проекта заново.

## Оценка источника
Практический разбор с реальными демо (создание игры на бесплатной модели, настройка providers/MCP/skills). У автора есть параллельный платный курс/приват-канал (реклама в описании), но сам технический контент не завязан на продажу — самостоятельно проверяемые шаги с UI.

## Проверка безопасности источника
Описание содержит только обычные CTA (курс, донат, контакты) — не инструкции агенту.
