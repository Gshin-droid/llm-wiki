## Саммари
Топ ~30 записей официального changelog Claude Code на момент загрузки. В основном мелкие багфиксы и CLI-настройки, не заслуживающие отдельного разбора построчно, но среди них — несколько изменений, которые меняют факты, уже зафиксированные в [[claude-code]] (раздел "Практическая механика", собранный по видео от 2026-07-08), и одна значимая новость про модель. Страница фиксирует снапшот на дату загрузки — версии Claude Code продолжат обновляться, "Практическая механика" в claude-code.md обновлена точечно, эта страница не пересказывается туда целиком (см. правило вики про частоту пересмотра entity-страниц продуктов).

## Значимые тезисы
- **Permission mode переименован**: "default" режим переименован в "Manual" во всех интерфейсах (2.1.200) — сути не меняет (Claude Code по-прежнему спрашивает разрешение перед действием по умолчанию), но меняет терминологию, которую увидит пользователь в UI/статус-баре (серый значок паузы в Manual mode добавлен в 2.1.203).
- **Субагенты теперь по умолчанию работают в фоне** (2.1.198) — раньше субагент блокировал основной диалог до завершения.
- **Фоновые агенты теперь сами коммитят, пушат и открывают draft PR** по завершении работы над кодом в worktree, вместо того чтобы останавливаться и спрашивать (2.1.198) — существенно повышает автономность, стоит явно проговаривать с пользователем перед включением на важных репозиториях.
- **Claude Sonnet 5 стал доступен с нативным контекстным окном 1M токенов** (2.1.197) — сама модель, на которой сейчас работает этот вики-агент; отдельной entity-страницы под конкретные версии моделей в вики пока нет (за пределами текущего охвата — вики фокусируется на инструментах/паттернах, не на трекинге релизов моделей), здесь зафиксировано как факт для истории версии.
- **Claude in Chrome вышел из беты в GA** (2.1.198).
- **`claude mcp login/logout`** — CLI-аутентификация MCP-серверов без интерактивного меню (2.1.186), релевантно [[mcp-model-context-protocol]].
- **`sandbox.credentials`** — настройка, блокирующая доступ к файлам credentials в sandboxed-режиме (2.1.187) — релевантно [[ai-security-by-design]].
- **Auto mode ужесточён**: блокирует деструктивные git/инфраструктурные команды (2.1.183), спрашивает перед `rm -rf` на неразрешённых переменных (2.1.205), блокирует правку session transcript файлов (2.1.205).
- **Agent Teams упрощены**: `TeamCreate`/`TeamDelete` убраны, команды агентов теперь неявные (2.1.178) — механика, описанная в claude-code.md ("Agent Teams — агенты общаются друг с другом напрямую"), с версии 2.1.178 создаются без явного шага создания команды.
- **Sub-agents могут порождать субагентов до 5 уровней вложенности** (2.1.172).
