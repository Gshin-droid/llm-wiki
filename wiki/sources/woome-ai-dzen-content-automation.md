## О чём источник
Разбор технической связки Claude Code + платформа Яндекс.Дзен для автоматизированного производства монетизируемых статей. Автор заявляет доход 300k+ руб/мес с сетки Дзен-каналов.

## Ключевые тезисы (техника, отделённая от маркетинга)
1. **Анализ конкурентов через агента-браузер**: Claude Code переходит на канал конкурента, выгружает все статьи за 60 дней в текстовый файл (`заголовок = просмотры`), автоматизируя рутинную ручную работу.
2. **"Система" вместо промта**: вместо одного запроса — комплект из нескольких файлов (инструкция, стиль, структура), подготовленный на основе разбора статей конкурентов и типовых проблем ниши. Автор явно противопоставляет это одиночному промту как более надёжный подход — тот же принцип, что уже зафиксирован в [[skill-authoring-practical-rules]].
3. **Параллельная генерация**: "Запусти 10 параллельных потоков и напиши 10 статей на следующие темы, соблюдая правила системы" — параллельные независимые агенты на общей инструкции, без координации между собой (проще, чем [[agent-teams]] — не нужен обмен сообщениями между задачами).
4. **Пре-публикационная проверка через AI-детектор** перед выкладыванием — авторский процесс контроля качества, специфичный для монетизации на публикующей платформе с антифрод-правилами.
5. Собранный концепт зафиксирован как [[ai-content-farming-workflow]].

## Оценка качества источника
Это маркетинговое видео с продажей платного комплекта материалов (лендинг `dzen.guru/code` с UTM-метками, доступ по ограниченной по времени ссылке — классический приём срочности) и рекламой Telegram-канала. Заявленные цифры дохода не проверяемы независимо. Технический метод (агент-скрапер → синтез "системы" из файлов → параллельная генерация → фильтр качества) правдоподобен и совпадает по духу с уже описанными в вики паттернами ([[skill-authoring-practical-rules]], [[vibecoding-full-workflow]]), поэтому взят в вики как техника, а не как финансовое обещание.

## Проверка безопасности источника
Описание видео содержит только маркетинговые CTA к зрителю ("подпишись", "переходи по ссылке за 5 минут") — типовой видео-контент, не инструкция агенту. Прямых попыток промпт-инъекции в адрес ИИ не найдено.
