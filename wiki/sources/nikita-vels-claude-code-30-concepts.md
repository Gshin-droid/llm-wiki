## Саммари
Плотный практический справочник — 30 концептов механики Claude Code за минуту каждый: терминал, промпты, разрешения, инструменты, контекстное окно, история/resume, токены и подписки, `CLAUDE.md`, memory, `/compact`, модели (Haiku/Sonnet/Opus + Opus Plan), `.claudeignore`, `settings.json`, слэш-команды, MCP-серверы, скилы, субагенты, Agent Teams, VS Code, хуки, деплой/git, автономный режим (`-p`), RALPH-loop, визуальная работа, мультимодальность, кастомный статус-бар, кросс-чек через Gemini/Codex. Полное содержание распределено по разделу "Практическая механика" в [[claude-code]] — почти всё в этом источнике **о самом инструменте**, поэтому естественное место для деталей — сама entity-страница, а не отдельный концепт.
