# На каком уровне владения Claude я сейчас (по данным этой вики)

**Вопрос:** на основе данных из вики — какой уровень по модели [[five-levels-of-claude-mastery]]?
**Оговорка:** оценка основана только на наблюдаемом использовании в рамках этого проекта (`C:\jarvis`) — не видно, как используется Claude.ai/Desktop в других задачах.

## Вывод: Уровень 4 (Продвинутый), к Уровню 5 пока не перешли

**Что совпадает с Уровнем 4:**
- Работа идёт через терминал/[[claude-code]], а не через чат в браузере — прямой маркер уровня
- Есть [[CLAUDE.md]], который реально накапливает правила по ходу работы (не написан один раз и забыт): в него уже добавлялись уточнения по специфике ИИ-домена, по проекту-как-отдельной-операции, разграничения по типам страниц — то есть файл эволюционирует так, как и описывает уровень 4
- Практика самопроверки уже присутствует в рабочем процессе: после правок вики автоматически прогонялась проверка на битые ссылки и страницы-сироты перед тем, как отчитаться о результате — это тот же паттерн "агент проверяет сам, прежде чем показать", что описан в уровне 4

**Чего не видно (пока не Уровень 5):**
- Нет облачных рутин по расписанию (например, автоматический периодический lint без явного запроса)
- Нет хуков (правил до/после действия агента)
- Нет управления через мессенджер
- Нет команд из нескольких агентов, которые спорят/критикуют друг друга
- Нет task budgets

**Нюанс:** модель уровней описывает прогресс через продукты Claude.ai/Desktop (Projects → Skills → Cowork), но здесь работа сразу пошла через [[claude-code|Claude Code]] в терминале, минуя браузерные вехи 2-3 уровня ([[claude-projects]], [[claude-skills]], [[claude-cowork]]) — то есть путь нетиповой, но по факту использования терминала и накопительного `CLAUDE.md` соответствует именно 4 уровню, а не ниже.

## Что могло бы стать следующим шагом к уровню 5
По модели уровней, переход 4→5 начинается с одной повторяющейся еженедельной задачи, вынесенной в облачную рутину. Здесь естественный кандидат — операция **Lint** (сейчас выполняется "раз в пару недель" по явному запросу) — её можно превратить в облачную рутину по расписанию, если/когда появится доверие отдавать это без ручного триггера.
