# Supabase

**Тип:** инструмент (Backend-as-a-Service)
**Актуально на:** 2026-07-07

## Что это
Готовый бэкенд: база данных (PostgreSQL), авторизация, файловое хранилище (buckets), serverless-функции (Edge Functions) и секреты — из коробки, без необходимости писать свой сервер. В вайбкодинге закрывает всю бэкенд-часть, пока разработчик описывает логику естественным языком через Claude/ИИ-агента.

## Роль в вайбкодинг-воркфлоу
См. [[vibecoding-full-workflow]] — Supabase хранит схему БД (SQL-миграции), файлы (buckets), секретные ключи (Edge Function secrets, чтобы не светить их на фронтенде).
