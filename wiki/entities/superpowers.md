# Superpowers

**Тип:** инструмент (open source набор скиллов для агентской разработки, TDD-ориентированный)
**Актуально на:** 2026-07-10

## Что это
Набор скиллов на разные этапы разработки (идея → спека → план → сборка (TDD) → проверка → финиш), устанавливаемый как plugin в Claude Code, Codex CLI, Gemini, OpenCode, Cursor. Легче и менее токен-затратный, чем [[gsd-get-shit-done]] — под небольшие/некорпоративные проекты.

Ключевой скилл — **`using superpowers`**: диспетчер, сам определяющий, какой скилл из набора нужен на текущем этапе (brainstorming для спеки, writing plans для декомпозиции, TDD для сборки, verification-before-complete, finish-development-branch). При исполнении плана может выбрать **subagent-driven execution** — самостоятельно создаёт именованных субагентов с явной ролью под конкретные подзадачи, работающих параллельно волнами; также автоматически подключает сторонние обнаруженные скиллы (пример: `frontend-design`, `deploy-to-vercel`), то есть работает как рамочная система с автообнаружением инструментов, а не жёсткий фреймворк.
