# OpenCode

**Тип:** инструмент (open-source CLI-агент для кодинга)
**Актуально на:** 2026-07-10

## Что это
Бесплатный open-source агент для кодинга, альтернатива [[claude-code]]. Ключевое отличие — мультипровайдерность (~100 провайдеров моделей, включая локальные Ollama/LM Studio/Hugging Face, тогда как Claude Code работает только с моделями Claude) и три интерфейса запуска (терминал, десктоп, веб на localhost).

Поддерживает ту же структуру Skills/MCP и файл-инструкцию `AGENTS.md` (аналог `CLAUDE.md`), встроенные режимы Plan/Build. Конфиг проекта — `opencode.json`, рекомендуется явно включить `ask` на редактирование файлов и bash-команды.
