# Claude Cowork

**Тип:** функция продукта (Claude Desktop)
**Актуально на:** 2026-07-07

## Что это
Режим Claude Desktop, в котором модель получает доступ к реальным папкам/файлам на компьютере пользователя (не просто к тексту в чате) и может действовать как "сотрудник за компьютером": сортировать файлы, использовать Computer Use (взаимодействие с интерфейсами без готовой интеграции), запускать задачи по расписанию (scheduled/облачные рутины), принимать задачи с телефона (Dispatch).

Соответствует 3-5 уровням в [[five-levels-of-claude-mastery]]:
- Уровень 3: доступ к файловой системе, Computer Use, Skills, управление с телефона
- Уровень 5 (Архитектор): облачные рутины независимо от включённого компьютера, хуки (правила до/после действия агента), команды агентов, task budgets

## Пример из источников
Разбор 2300+ файлов в папке загрузок по типам за несколько минут (см. [[qaisar-claude-full-course]]); еженедельная автоматическая уборка по расписанию.
