# Pinecone

**Тип:** инструмент (векторная БД, freemium)
**Актуально на:** 2026-07-07

## Что это
Векторная база данных для embedding-поиска. В [[hybrid-memory-architecture|гибридной архитектуре]] играет роль "склада" — абсолютной статичной памяти: многочасовые транскрипты, архивы переписок, датасеты. В отличие от Obsidian (для того, что постоянно меняется), Pinecone — для данных, которые не меняются: векторизация делает поиск по ним семантическим и дешёвым при масштабе, без участия Claude в процессе индексации.

## Статус в этой вики
Не используется — вики только создана, объём источников пока минимален. Стоит вернуться к этому вопросу, когда `raw/` разрастётся.

## Уточнение по более предметному источнику
[[llm-memory-landscape]] (более техничный обзор) вообще не называет Pinecone среди актуальных решений для памяти агента — там фигурируют [[mem0]], [[zep-graphiti]], [[letta]] как более специфичные инструменты. Pinecone — общий примитив векторного поиска, который может использоваться *внутри* таких фреймворков, а не самостоятельное решение "для масштабирования". Формулировка "Pinecone для больших архивов" из [[mastodont-claude-obsidian-video]] стоит считать упрощением, актуальным на момент того видео, а не точной рекомендацией.
