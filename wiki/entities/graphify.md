# Graphify

**Тип:** инструмент (локальный граф кодовой базы)
**Актуально на:** 2026-07-07

## Что это
Решает другую задачу, чем memory-фреймворки выше — не "помнит факты о пользователе", а строит граф архитектуры кода: NetworkX (in-memory граф) + Leiden clustering (поиск связанных файлов "одной фичи") + tree-sitter (AST-парсинг 25+ языков, структурные связи call/import/inheritance без LLM, за 0 токенов) + vis.js (интерактивная HTML-визуализация). Ключевая фишка для Claude Code — PreToolUse hook: перед каждым Glob/Grep агент сначала видит one-page обзор архитектуры (god nodes, кластеры), а не бежит по файлам вслепую.

## К заявленным цифрам — скепсис
Маркетинг обещает до 71.5× экономии токенов — это лучший случай в синтетическом сценарии, не средняя цифра в проде. Реалистичнее 2-5×, до 10× в задачах, где граф точно ложится по структуре ответа. Автор Graphify строит коммерческий продукт (Penpax) поверх open-source версии — стоит учитывать экономический интерес при чтении заявленных метрик.
