# Mem0

**Тип:** инструмент (memory framework / библиотека)
**Актуально на:** 2026-07-07

## Что это
Библиотека памяти для собственных ИИ-агентов на API (не для пользователей готовых продуктов вроде Claude.ai). Извлекает факты из сообщений через LLM, хранит в векторной базе, применяет ADD/UPDATE/DELETE/NOOP при новых фактах. В свежих версиях упрощена до однопроходного ADD-only + entity linking, поддержка графовых хранилищ убрана.

## Ограничения
Нет временной модели (может перепутать старый и новый факт при противоречии), embedding-поиск не всегда склеивает семантические дубликаты на больших объёмах.

## Когда уместен
Свой агент на API, нужна персонализация под пользователя, десятки тысяч фактов без экзотики. Не нужен, если хватает встроенной памяти Claude/ChatGPT как продукта.
